Jeedom se puede instalar en diferentes componentes de hardware: 

- una Raspberry pi 1 o 2
- una Jeeboard (hummingboard) i1, i2 o i4
- un nas Synology
- todos los sistemas operativos Linux (basado en Debian de preferencia)
